# SPL (SimplePatreonLoader) Vimeo Video Loader

## About The Project

SPL is Tampermonkey/Greasemonkey userscript designed to simplify video loading on Vimeo.



## Getting Started

### Prerequisites

- Web Browser (Chrome, Firefox, etc.)
- Tampermonkey (Chrome) or Greasemonkey (Firefox) extension

### Installation

1. Install Tampermonkey (Chrome) or Greasemonkey (Firefox)
   - [Tampermonkey for Chrome](https://chromewebstore.google.com/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo)
   - [Greasemonkey for Firefox](https://addons.mozilla.org/en-US/firefox/addon/greasemonkey/)

2. Create a New Userscript
   - Open Tampermonkey/Greasemonkey
   - Click "Create a new script"
   - Copy and paste the entire script from the `spl-vimeo-loader.js` file

3. Save the Script
   - Press Ctrl+S (Cmd+S on Mac)
   - Ensure the script is enabled in the Tampermonkey/Greasemonkey dashboard

### Usage

1. Navigate to any Vimeo video page
2. The script will automatically attempt to load the video in a full-screen iframe

## Compatibility

- Requires Tampermonkey or Greasemonkey
